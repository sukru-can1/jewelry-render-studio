## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → zip route | Untrusted batch id + ?deliverables; layer/deliverable set derived from DB + deterministic pathnames, never caller input |
| private Blob → zip stream | Deliverable bytes repackaged (not decoded) into the zip |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-09 | Spoofing | unauth batch zip | mitigate | requireSession() first line (inherited from the existing route) |
| T-06-10 | Elevation/IDOR | zip another project's deliverables | mitigate | batch loaded by params.id; deliverable pathnames derived from that batch's DB variants only |
| T-06-11 | Information Disclosure | public/signed deliverable URL | mitigate | get(...,{access:'private'}) only; blob-guard source guard covers the route |
| T-06-12 | DoS | whole-batch synchronous flatten overruns 60s | mitigate | cap lazy flattens at N per request + stream (never buffer) the zip; remainder noted in a header |
</threat_model>
